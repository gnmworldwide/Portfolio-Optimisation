# Portfolio Optimisation (Mean-Variance, Monte Carlo)
A compact portfolio optimisation project using a Markowitz-style mean-variance framework and a Monte Carlo search for the maximum Sharpe ratio.

## What It Does
- Generates synthetic daily returns for multiple assets
- Annualises expected returns and the covariance matrix
- Randomly generates long-only portfolios (weights sum to 1)
- Selects portfolio with highest Sharpe ratio

## Requirements
- Python 3.9+
- numpy

## Usage
python optimiser.py

## Extensions
- Use real price data
- Add portfolio constraints
- Add transaction cost models
